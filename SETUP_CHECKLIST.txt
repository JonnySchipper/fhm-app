========================================
TROUBLESHOOTING
========================================

Problem: "Images folder not found"
Fix: Put FHM_Images in parent directory, not inside this folder

Problem: "Image not found for character"
Fix: Check character name matches image filename exactly

Problem: "Template PDF not found"  
Fix: Make sure format.pdf is in same folder as script

Problem: "Font not found"
Fix: Verify font\ folder exists with all 3 font files
